# Music and Image Tagging

This project contains scripts for managing and synchronizing [Deezer](https://www.deezer.com/de/) playlists and [Eagle](https://eagle.cool/), as well as tools for correcting date information.

## Motivation

I originally used Eagle to manage all my Photos and Videos, because of the tagging system. Tagging is a great alternative to storing files in folders.

The idea was to apply this same concept to my music collection, and since Deezer has an API that allows you access to public playlists, I could import mine, and save them in Eagle as album covers. By adding the URL to the album covers, you can then jump back to Deezer, and listen to the tracks.

It is really useful to see which tracks are in multiple playlists, and I have greatly improved my library's structure since then.

## Requirements

- Python
- Eagle app running locally
- Deezer account/user id

## Usage

1. Set up .env with required variables (see scripts for details).
2. Run scripts.
